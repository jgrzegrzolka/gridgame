# Flag Grid

A geography grid puzzle game. Each cell requires a country whose flag satisfies both the row and column criteria.

## Local dev

```
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys to GitHub Pages automatically.

## Enable GitHub Pages (one-time setup)

1. Go to repo → **Settings** → **Pages**
2. Under "Source" select **GitHub Actions**
3. Save

That's it. Next push deploys automatically.
